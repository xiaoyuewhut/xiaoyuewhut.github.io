# 传感器种类
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205114453404.png)

概括一下：
| 传感器类型   | 主要功能  | 典型应用   | 工作频率/范围  |
| --------------- | --------- | ------------ | -------------- |
| **摄像头**   | 图像识别、物体分类 | 车道偏离预警、交通标志识别、行人检测、盲点监测、360°环视   | 前视(150m)、环视(20m)   |
| **毫米波雷达**  | 距离、速度测量   | 自适应巡航(ACC)、自动紧急制动(AEB)、盲点监测(BSD)、变道辅助(LCA) | 77GHz（长距离）、24GHz（短距离） |
| **激光雷达(LiDAR)** | 高精度三维环境感知 | 障碍物检测、道路识别、导航规划   | 长距离高精度扫描 |
| **超声波雷达** | 近距离障碍物探测  | 倒车雷达、自动泊车(APA)、低速碰撞预警 | 短距离（通常<5m）|

>一般来说，汽车都会装以下几个传感器：

|传感器|数量|
|--|--|
|前视摄像头|1|
|77GHz 前向毫米波雷达|1|
|24GHz 侧向毫米波雷达|4|
|超声波雷达|12|
|环视鱼眼摄像头|4|

## 为什么前向毫米波雷达采用77GHz、侧向采用24GHz？
主要是基于探测距离、精度需求、成本控制和安装位置等因素的综合考量。

> 目前77GHz也正逐步向侧向应用渗透

前向用77GHz的原因：
1. **探测距离**
前向雷达主要用于**自适应巡航(ACC)**、**自动紧急制动(AEB)**、**前向碰撞预警(FCW)** 等纵向控制功能，需要探测远处目标（100-250米），为系统预留充足反应时间。77GHz雷达的探测距离可达100-250米，满足高速场景需求。

2. **精度需求**
77GHz频段可提供高达4GHz的**扫描带宽**，而24GHz的ISM频段仅200MHz带宽。带宽越大，距离分辨率越高，能更精确**区分相邻目标**；77GHz波长更短，速度测量性能比24GHz提高3倍，可生成**更高分辨率的距离-速度图像**，更适合识别动态目标。

侧向用24GHz的原因：
1. **中短距离探测需求**
侧向雷达主要用于盲点监测(BSD)、变道辅助(LCA)、后方交叉交通预警(RCTA) 等功能，探测距离要求较短（通常30-70米），24GHz雷达30-60米的探测范围已能满足需求。

2. **成本低**

3. **宽视角覆盖**
侧向雷达需要±70°至±80°的宽视角来监测相邻车道和侧后方区域。虽然24GHz精度较低，但其天线设计更易实现宽视场角覆盖，满足侧向监测的广度要求。

目前主流采用  "1+2+2"布局  ：
+ 前向：1颗77GHz长距离雷达（ACC/AEB）
+ 侧后向：4颗24GHz中短距雷达（BSD/LCA/RCTA）

例外：阿维塔 07 就只用了 3个毫米波雷达，应该是1颗前向+2颗后角雷达。

## 为什么 ACC/AEB 通常会融合前视摄像头和毫米波雷达？
| 传感器类型     | 核心优势      | 主要短板    | 互补效果    |
| --------- | ------------- | ------------------------ | -----------|
| **毫米波雷达** | - 精确测距测速（误差<0.5米）<br>- 全天候工作（穿透雨雾）<br>- 抗光线干扰强      | - 分辨率低，难以区分相邻目标<br>- 无法识别物体类型和颜色<br>- 对静止目标易误判 | 提供目标的**距离、速度、方位**基础数据   |
| **摄像头**   | - 丰富的图像纹理信息<br>- 精准识别物体类别（车/人/标志）<br>- 检测车道线、交通标志颜色 | - 受光照、天气影响大（夜间/逆光/浓雾性能下降）<br>- 深度信息缺失，测距精度低    | 提供目标的**类别、尺寸、车道归属**语义信息 |

> 例如：雷达发现"有物体在50米处接近"，摄像头识别"是一辆货车且在相邻车道"，融合后系统才能做出准确决策——不减速，避免单一雷达因无法分类而产生的误刹车。

**所以总体上的目标是为了增强感知能力，而且单一传感器容易丢失目标。**

另外就是**基础ACC仅需雷达即可实现跟车，但融合摄像头后可支持 Highway Assist（高速驾驶辅助） 和 Traffic Jam Assist（拥堵辅助） 等高级功能。**

因为一个先进的 ACC 不止有跟车和巡航的功能需求，还要有比如 Stop&Go 等需求，这些也是 ACC 需要做的，单靠雷达是有**静止目标识别差**的，但是融合摄像头可以识别前车轮廓，实现全速域跟停再起步。

如果更进一步，比如做 LCC、NOA 等，就需要识别车道线、红绿灯等，单靠雷达无法做到。

(融合方案通常是后融合或特征级融合，部分领先方案会采用前融合，直接对齐雷达点云和像素坐标)


# 安装位置及主要参数
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205123441638.png)
## 前向毫米波雷达
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205123939596.png)
一般是安装在**车辆前部中央区域**，比如：
+ 前保险杠内侧
+ 车标后方
+ 车头栅格内

具体安装技术要求如下：
+ **高度**：补偿车辆负载后，400-800mm
+ **角度**：水平特征与路面平行，垂直特征与路面垂直，pitch/roll公差控制在  ±4.5°  内
+ **与覆盖件距离**：20-30mm
+ **覆盖件要求**：厚度均匀、表面平整（平整度<400μm），喷漆不超过4层且不含金属成分

**典型配置**：
+ 探测范围：200米以内
+ 水平视场角：±45°
+ 垂直视场角：通常较窄（约±5°）

### 前向毫米波雷达是否必须安装在中轴线上？
> 推荐但不必要。

安装在中轴线上可以保证左右探测范围对称，减少一些盲区；也便于和车辆坐标系对齐，后续的坐标转换复杂度降低；

但是允许左右偏差大概 600 mm 以内，并且角度误差造成的后果会比位置误差更严重。

## 侧向毫米波雷达（角雷达）
安装位置：**车辆四个拐角处**
+ 前保险杠左右角：监测侧前方盲区
+ 后保险杠左右角：监测侧后方及后方交叉交通

**典型配置**：
+ 探测距离：30-70米
+ 水平视场角：±60°至±80°（广角覆盖相邻车道）

## 超声波雷达
超声波雷达也分为两种：**短程**和**远程**。
| 类型          | 安装位置  | 探测距离     | 主要功能              | 典型数量     |
| ----------- | ----- | -------- | ----------------- | -------- |
| **UPA（短程）** | 前后保险杠 | 15-250cm | 倒车雷达(PDC)、低速防撞    | 前4+后4=8颗 |
| **APA（远程）** | 车身侧面  | 30-500cm | 自动泊车(APA)、侧方障碍物检测 | 左右各2颗    |

一般用12颗雷达（前4+后4+侧4）可实现APA自动泊车功能
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205170010998.png)

**前后保险杠各 4 颗**，其中中部两颗负责正前方/正后方中距离探测（30-250cm），两角各 1 颗覆盖侧前方/侧后方盲区（避免转弯、泊车时与邻车或路沿碰撞）。

前 4 颗：
+ 位置：应当安装在保险杠内侧，Logo两侧；
+ 高度：补偿车辆负载后，300-500mm（不宜过高以免探测不到低矮障碍物，不宜过低易受泥水覆盖）；
+ 间距：探头间距30-40cm，确保探测范围重叠。

后 4 颗：
+ 位置：后备箱盖或后保险杠上；远离排气管、大功率灯具等热源；需覆盖车尾及侧后方盲区


**左右侧裙各两颗**，沿车门下方布置，实时监测侧方距离，用来识别车位边界、邻车间距，防止侧擦。
+ 位置：车门下方侧裙或车身侧面
+ 特殊要求：沿车身长度方向布置，确保能检测车位边界

## 环视摄像头
环视摄像头采用  "四目布局"  方案，通过四颗广角摄像头分别安装在车辆前、后、左、右四个方向，采集图像后经算法拼接成360°鸟瞰图，为驾驶员提供无盲区视野。

| 摄像头位置     | 典型安装点         | 主要功能             | 视角要求        |
| --------- | ------------- | ---------------- | ----------- |
| **前置摄像头** | 前格栅、车标或前保险杠中央 | 捕捉前方低矮障碍物、停车通道标识 | 180°-200°广角 |
| **后置摄像头** | 尾门牌照灯附近、后保险杠  | 倒车影像、后方障碍物检测     | 180°-200°广角 |
| **左视摄像头** | 左侧后视镜根部内侧     | 捕捉左侧盲区、路边沿距离     | 180°-200°广角 |
| **右视摄像头** | 右侧后视镜根部内侧     | 捕捉右侧盲区、路边沿距离     | 180°-200°广角 |

![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205170925891.png)

> 需要确保车辆周围任意物体至少被2个摄像头同时捕捉，为拼接算法提供足够的重叠区域，消除视觉盲区。

1. **前置摄像头**
安装高度：通常离地400-600mm（与毫米波雷达高度相近）
覆盖件要求：前格栅或车标区域需为非金属材料，避免遮挡信号

2. **后视镜摄像头**
安装位置：集成于后视镜外壳根部内侧，避免雨水和泥污直接溅射
角度调整：需精确校准，确保左右摄像头光轴与车身纵向平行
防护设计：镜头表面需防水防污，部分车型配备加热除雾功能

3. **后置摄像头**
安装位置：尾门右侧牌照灯附近或后保险杠中央
高度匹配：与前置摄像头保持相近高度，确保拼接后图像比例一致

> 部分大型SUV或MPV会增加两颗摄像头，分别安装在前翼子板和后轮眉。[6路1080P车载360全景影像系统](https://cn.stonkam.com/products/360-bird-view-system-BVA18P.html)

> 对于一些无实体后视镜的车，或者将来想取消实体后视镜的车，侧视摄像头会集成于翼子板而非后视镜。

## 激光雷达
目前激光雷达主要有三种安装方式：车顶方案、前格栅/保险杠方案、车灯集成方案。

其余还有一些辅助布局：
| 安装位置     | 探测范围    | 主要功能        | 适用场景        | 外观影响 |
| -------- | ------- | ----------- | ----------- | ---- |
| **车顶**   | 远距离360° | 全局环境扫描、高速巡航 | 高速NOA、城市NOA | 较大   |
| **前格栅**  | 中近距离前向  | 低矮障碍物、精确泊车  | 自动泊车、低速穿越   | 无    |
| **车灯**   | 中远距离前向  | 前向补盲、美观集成   | 高速+泊车       | 无    |
| **侧向**   | 中距离侧向   | 盲区监测、并线辅助   | 变道辅助、十字路口   | 较小   |
| **后向**   | 中距离后向   | 倒车监测、后方变道   | 自动泊车、高速变道   | 无    |
| **挡风玻璃** | 中近距离前向  | 冗余参考、隐蔽探测   | 全场景辅助       | 无    |


1. **车顶方案**
一些主流高端车型都用这个方案，比如理想L9、蔚来ET7、阿维塔11。
一般将**机械旋转式或混合固态激光雷达**安装在**车顶中央或偏前部位**，视野开阔，会有个小犄角，所以有的人觉得影响美观。
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/20251205172556576.png)

2. **前格栅/保险杠方案**
一般采用**固态或小型机械激光雷达**，安装在**前保险杠内侧或格栅后方**，用于补充车顶雷达对路缘石、地面积雪/水坑、停车桩等低矮目标的盲区。
![](https://cdn.jsdelivr.net/gh/xiaoyuewhut/image@main/641)

3. **车灯集成方案**
一般采用**微型固态激光雷达**，嵌入前大灯组内部。小鹏G6、G9等车型已采用此方案。（维修成本会很高，而且大灯会有热量问题）

**布局总结**
高阶自动驾驶（L2+及以上）通常采用 **"1+2+4"** 组合：
+ **1颗顶置长距雷达**：负责前向远距离全局感知
+ **2颗前角雷达**：补充低矮障碍物和侧前方盲区
+ **4颗侧后雷达**：实现360°环视覆盖
